Open-Letters Newslettersystem - Aktualisierungen/Updates

letzte Änderung: 2014-12-30, Stefan Rank-Kunitz

Übersicht:

Das Einspielen von Aktualisierunge des Open-Letters Newslettersystems gliedert sich, ähnlich der Installation, in diese Schritte:
  • Sicherung der bestehenden Installation
  • Upload der Dateien und Verzeichnisse (per FTP) Verzeichnis-Rechte setzen
  • Datenbank aktualisieren
  • Konfiguration anpassen

a) Sicherung (Backup) des bestehenden Systems:

Zu Beginn sichern Sie bitte die bestehende Installation des Newsletter-Systems auf Ihrem PC: Laden Sie zuerst alle Dateien und Verzeichnisse mit Hilfe eines FTP-Programmes von Ihrem Webspace herunter auf Ihren PC. Bitte stellen Sie dabei sicher, dass auch die Dateien .htaccess und .htpasswd aus dem Verzeichnis admin mit heruntergeladen wurden => manche FTP-Programme ignorieren mit einem Punkt beginnenden Dateien.

Darüber hinaus sichern Sie bitte die Datenbank (und damit alle im Newsletter-System gespeicherten Daten) über die Datenbank-Verwaltung, die Ihnen Ihr Webhoster zur Verfügung stellt. Das häufig angebotene Werkzeug "phpMyAdmin" bietet über den Menüpunkt "Exportieren" die Möglichkeit, die gesamte Datenbank in eine SQL-Datei zu speichern und diese herunterzuladen.
(Hinweis: Ebenso bietet der Menüpunkt "Importieren" die Möglichkeit, die heruntergeladene SQL-Datei wieder hochzuladen und damit den gesicherten Stand der Datenbank wiederherzustellen.)

Bitte bewahren Sie die heruntergeladenen Daten an einem SICHEREN Ort auf: Die per FTP heruntergeladenen Dateien und Verzeichnisse enthalten unter anderem Ihren Zugang zur Admin-Oberfläche, in der gesicherten SQL-Datenbank sind die Namen und E-Mail-Adressen aller bei Ihnen registrierten Newsletter-Empfänger enthalten.

b) Download, FTP-Upload, Rechte setzen:

Laden Sie nun die neuste Version des Newslettersystem von unserer Website http://www.open-letters.de herunter. Entpacken Sie die Zip-Datei. Löschen Sie die folgenden Verzeichnisse von Ihrem Webspace und laden Sie diese aus der entpackten Zip-Datei neu hoch.

  • admin
  • external_scripts
  • framework
  • include
  • languages
  • index.php
  • newsletter_view.php

Hinweis: Das reine "Überspielen" der Dateien und Verzeichnisse führt häufig nicht zum gewünschten Ergebnis. Bitte löschen Sie die aufgelisteten Dateien und Verzeichnisse weg und laden Sie diese neu hoch.

Nach dem Hochladen des Verzeichnisses admin stellen Sie bitte den htaccess-Verzeichnis-Schutz wieder her. Laden Sie dazu die Dateien .htaccess und .htpasswd aus Ihrem oben erzeugten Backup wieder in das Verzeichnis admin.

Bitte prüfen Sie anschließend die Verzeichnisrechte, wie im Abschnitt a) der INSTALLATION beschrieben ist.

c) Datenbank-Update:

Öffnen Sie Ihren Webbrowser und loggen Sie sich in die Admin-Oberfläche des Newsletter-Systems ein. Rufen Sie dann die Datei update.php, welche ebenfalls im Verzeichnis admin liegt: Haben Sie das Newslettersystem unter der Internetadresse http://www.example.com/ installiert, so starten Sie das Updateprogramm update.php, indem Sie die Internetadresse http://www.example.com/admin/update.php mit Ihrem Browser öffnen.

d) Anpassung der Konfiguration:

Seit Ihrer Erst-Installation des Open-Letters Newsletter-Systems kann es sein, dass in Folge funktionaler Erweiterungen oder Problembehebungen Änderungen an der Konfiguration erforderlich geworden sind.
Wir empfehlen Ihnen, dass Sie die auf Ihrem Webserver verwendete Datei config/config.inc.php mit der nun neu heruntergeladenen Version vergleichen. Bitte prüfen Sie, ob neue Konfigurationen hinzu gekommen sind, andere entfallen oder sich die Verwendung von Einstellungen verändert hat.

Bearbeiten Sie die Datei config.inc.php im Verzeichnis config mit einem Editor, z.B. dem Programm "Geany" von http://www.geany.org Bitte benutzen Sie dazu kein Text-Verarbeitungsprogramm wie Microsoft Word oder das LibreOffice oder OpenOffice.org! Laden Sie die Konfigurations-Datei nach eventuellen Änderungen neu mit einem FTP-Programm auf den Webserver hoch.
